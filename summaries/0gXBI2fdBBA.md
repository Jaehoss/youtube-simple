## 상세 내용

### 1. 매드맥스 시리즈 = 시대를 반영한 예언적 영화

- **매드맥스 1 (1979):** 저예산($35만) 고속도로 복수극. 시대 반영보다는 거친 매력으로 흥행
- **매드맥스 2 (1981):** 이란혁명 + 이란-이라크 전쟁 -> 2차 석유파동 -> "석유 없는 세상" 묘사. 기름 한 방울을 위해 살인하는 세계관
- **매드맥스 3 (1985):** 체르노빌, 핵 공포 시대 -> "핵전쟁 이후 세상" 묘사. 2003~2005년 핵전쟁 후 호주를 배경으로 함

### 2. 1979년 -- 세계사의 전환점 (한 해에 벌어진 일들)

- 이란 종교혁명 (시아파 신정국가 탄생)
- 미국-중국 정식 수교
- 소련 아프가니스탄 침공
- 한국 10.26 사태 (박정희 대통령 암살)
- 이듬해 1980년 이란-이라크 전쟁 발발 -> 2차 석유파동

### 3. 이란-이라크 전쟁의 배경

- 이란 종교혁명 -> 이라크 시아파가 동조할까 우려한 사담 후세인이 선제공격
- 바트당(세속주의) vs 이란(신정주의)의 대리전 성격
- 터키도 세속주의 편 -> 이란이 고립된 채 전쟁
- 유전 파괴가 일상 -> 원유 공급 위기 심화

### 4. 한국의 핵개발 (890 계획)

- 1969년 닉슨 독트린 -> 주한미군 7사단 철수 -> 안보 위기
- 1974년부터 박정희 대통령이 핵개발 추진
- 이위소 박사 귀국 (1974.9) -> 교통사고 사망 (음모론 존재)
- 미국 압력으로 공식 포기했으나, 비밀리에 계속했다는 설
- 1974년 육영수 여사 피살, 1979년 박정희 암살 -- 핵개발과의 연관성 음모론

### 5. 이스라엘의 핵 보유와 "핵 모호성" 전략

- 네게브 사막 디모나 지역에 원자력 연구센터 (프랑스 기술 지원)
- 미국에 "섬유 공장"이라 거짓말 -> 미국이 속아줌
- 1960년대 사실상 핵보유 인정
- 1986년 내부 기술자 모르데카이 바누누가 폭로 -> 18년형 선고
- **핵 모호성 전략:** 인정도 부정도 않고, 누구도 묻지 않는 암묵적 합의
- 삼손 전략: "우리가 무너질 바에는 같이 죽자" (4차 중동전쟁 시 핵 사용 준비 지시)
- 현재 약 90기 핵탄두 보유, 최대 400기 분량의 플루토늄 보유 추정

### 6. 공식 핵보유국 vs 비공식 핵보유국

- **공식 (NPT 인정, 5개국):** 미국, 러시아, 중국, 영국, 프랑스
- **비공식 (4개국):** 인도, 파키스탄, 이스라엘, 북한
  - 인도/파키스탄: NPT 미가입, 둘이 싸울 때 아무도 개입 안 함
  - 핵 보유국끼리 싸우면 아무도 개입하지 않는 현실

### 7. 핵 금기의 붕괴 -- 최근 발언들

- **이스라엘** (2023): 문화유산 장관 "가자에 핵폭탄 투하도 선택지" (발언 후 해임)
- **미국** (2026.3.14): 데이빗 삭스 AI/암호화폐 차르 "이스라엘이 밀리면 핵 사용 가능"
- **러시아** (2024): 핵 사용 규칙 개정 -- 재래식 공격이라도 영토 침입 시 핵 사용 가능
- **북한** (2022): "지도부 위험 시 자동 핵타격" 선언
- 과거에는 "있어도 없는 것처럼" 다루던 핵을 이제 공개적으로 언급하는 시대

### 8. 전술핵의 위험성

- 전술핵 = 소형 핵무기, 좁은 지역만 타격 가능 (수십km 범위)
- 아직 한 번도 실전 사용된 적 없음
- 위험 시나리오: 한쪽이 전술핵 사용 -> 상대도 전술핵 대응 -> 에스컬레이션 -> 전략핵(대형) 사용
- "한번 쓰이면 주식이 문제가 아닌 시대가 될 수 있다"

### 9. 결론: 열전의 시대 회귀

- 미국 쉐일 혁명 -> 중동 석유 의존도 감소 -> 미국이 세계 질서 관리 의지 약화
- 다극 체제 전환 -> 지역 전쟁 증가 (러-우, 이란-이스라엘, 아시아?)
- 2차대전 이후 ~80년간 대규모 열전 없었지만, 무기 기술은 계속 발전
- "인간은 평화를 잘 못 견딘다" -> 다시 열전 시대로 진입 중

### 투자 힌트 (영상 내 암시)

- 방산주 (드론 제조사)
- 원전 건설사
- 직접 종목 언급은 피했지만, 차트 좋은 관련주를 찾아보라는 조언

## 핵심 메시지

> 석유 위기에서 핵 위협으로, 다시 열전의 시대로 -- 매드맥스가 예언한 미래가 현실이 되고 있다. 쉐일 혁명으로 석유 위기(매드맥스 2)는 피했지만, 핵 금기의 붕괴(매드맥스 3)는 현재 진행형이다.
